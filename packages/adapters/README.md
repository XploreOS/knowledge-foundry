# @knowledge-foundry/adapters

Ingestion and format adapters for [Knowledge Foundry](https://github.com/XploreOS/knowledge-foundry): local filesystem, web, Markdown, JSON, CSV, and PDF (stub). Used by `@knowledge-foundry/cli`'s `kf ingest` / `kf normalize`; install this package directly only when embedding the pipeline in your own tooling.

```bash
npm install @knowledge-foundry/adapters
```
